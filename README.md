# Simple NFT Viewer

This is a simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet
- Detect NFT smart contracts
- Show NFT token IDs
- Learn how NFTs work

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- MetaMask wallet
- Internet connection
- Basic JavaScript knowledge

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Click "Connect Wallet"
5. Click "Load NFTs"

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Network settings  
- `wallet.js` = Wallet connect  
- `nft.js` = NFT reader  
- `ui.js` = Screen updates  
- `abi.js` = NFT contract ABI  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs
